Note:

  • There are "controlled" versions of many operators such as controlledSwap, controlledX, controlledY, etc. For all of these, the controlBits must be specified before the target bits. (There is presently no controlled version of qft).
  • Most of the bit specifiers can be single numbers (0 being least significant), arrays of numbers, bit ranges (e.g. {from: 5, to:15}, or the constant jsqubits.ALL. Currently, the exception to this is applyFunction as this cannot take arrays. Also, while the controlledSwap method is able to take the various types of bit specifiers as control bits, it (and the swap) operator can only take single numbers for the two bits to be swapped.

For more details, see the jsqubits user manual.
